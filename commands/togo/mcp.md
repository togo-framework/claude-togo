---
description: Inspect / wire the togo MCP server (generators + project introspection as tools).
---

The **togo** MCP is auto-connected by this plugin (`https://mcp.to-go.dev/mcp`). Use it for project introspection and generator tools. To (re)wire MCP into a specific app or another agent locally, run `togo mcp:install --agent claude-code`; `togo mcp:list` / `togo mcp:test` inspect the wiring. Summarize the available MCP tools for the user.
