description: Check the togo toolchain (Go, Node/npm, sqlc, Atlas) and auto-install what's missing.
---

Run `togo doctor`. It verifies the toolchain — Go, Node/npm, sqlc, Atlas — and reports versions, installing anything missing. If the `togo` CLI itself isn't found, install it: `curl -fsSL https://to-go.dev/install.sh | sh`. Summarize the results and any actions taken.
